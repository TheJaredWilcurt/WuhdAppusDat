Install Instructions

  • Windows
    • Download the Installer and run it
    • It will default to installing to Program files, but you can set the folder anywhere (the app is portable)
    • After the install completes, it should pop open the app automatically
    • There will also be a shortcut in your start menu
  • Linux
    • Download the zip file and unzip it to any directory you like
    • In your terminal, cd into it and run ./WuhdAppusDat to run the program
    • If it asks for you to create a keychain, you can do so or ignore it. I have no idea why it asks that, Linux is weird.
  • OSX
    • Download and unzip the file. You'll probably want to put it in your Applications folder.
    • You may need to allow running a program from the internet.
    • You will need to allow WuhdAppusDat to have access to "Accessibility Features" (this is how it can see the title of the currently focused window).

Upgrading from Beta

  • If you see this popup when opening v2.0.0+, follow these instructions.
  • Close WuhdAppusDat (if running)
  • Delete the app cache folder (will also delete your settings, but the settings values changed between v1 and v2 anyway)
    • Windows: C:\Users\<YOUR USER NAME>\AppData\Local\wuhdappusdat
    • Linux: ~/.config/WuhdAppusDat
    • OSX: ~/Library/Application Support/WuhdAppusDat/Default
  • You should only have to do this once
  • Open WuhdAppusDat and adjust settings

Made With

NW.js is what allows WuhdAppusDat to run as a desktop app on each platform. It's super easy to use.

The application and interface logic is written using Vue.js. I specifically re-wrote the app from VanillaJS to Vue to save time on adding new features. Without it, there would never have been a version 2.
